# Heart disease training data

Training data for a binary prediction model (here: FFT) on (a subset of)
the
[`heartdisease`](https://www.nathanieldphillips.co/FFTrees/reference/heartdisease.md)
data. The complementary subset for model testing is
[`heart.test`](https://www.nathanieldphillips.co/FFTrees/reference/heart.test.md).
The data in
[`heartdisease`](https://www.nathanieldphillips.co/FFTrees/reference/heartdisease.md)
contains both subsets.

## Usage

``` r
heart.train
```

## Format

A data frame containing 150 rows and 14 columns (see
[`heartdisease`](https://www.nathanieldphillips.co/FFTrees/reference/heartdisease.md)
for details).

## Source

<https://archive.ics.uci.edu/ml/datasets/Heart+Disease>
