# Heart disease testing data

Testing data for a
[`heartdisease`](https://www.nathanieldphillips.co/FFTrees/reference/heartdisease.md)
data. This subset is used to test the prediction performance of a model
trained on the
[`heart.train`](https://www.nathanieldphillips.co/FFTrees/reference/heart.train.md)
data. The dataset
[`heartdisease`](https://www.nathanieldphillips.co/FFTrees/reference/heartdisease.md)
contains both datasets.

## Usage

``` r
heart.test
```

## Format

A data frame containing 153 rows and 14 columns (see
[`heartdisease`](https://www.nathanieldphillips.co/FFTrees/reference/heartdisease.md)
for details).

## Source

<https://archive.ics.uci.edu/ml/datasets/Heart+Disease>
