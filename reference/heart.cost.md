# Cue costs for the `heartdisease` data

This data further characterizes the variables (cues) in the
[`heartdisease`](https://www.nathanieldphillips.co/FFTrees/reference/heartdisease.md)
dataset.

## Usage

``` r
heart.cost
```

## Format

A list of length 13 containing the cost of each cue in the
[`heartdisease`](https://www.nathanieldphillips.co/FFTrees/reference/heartdisease.md)
dataset (in dollars). Each list element is a single (positive numeric)
value.

## Source

`https://archive.ics.uci.edu/ml/machine-learning-databases/heart-disease/costs/`
